Part 1:Technological Empathy: high-level understanding of how the Internet works

Understanding of how users, computers, servers, http, and the Internet all fit together:

• HTML: Main type of documents on the Web
• URLS: How to refer to documents on the web
• HTTP: Protocol that unites the web
• HTML: hypertext markup language - the basis for almost every webpage (the glue that binds everything together).

Anything can exist on the web: plain text, html, images, video, and music.

Major pieces:

• Your browser makes requests to servers via the Internet.
• The requests use a protocol called HTTP and the servers responds with files that your browser displays.
• Severs are optimized for hosting files.

HTML documents are the heart of the web, which is made of:

• Text content: what you see
• Mark up: what it looks like
• References to other documents (like) image and videos
• Links to other pages

HTML is made of < tags >:

Example:

< name > contents < /name >

< opening tag > contents < closing tag >

First tag to learn is the < b > or bold tag:

< b > contents < /b >

em tag = emphasis = italics

< em > contents < /em >

HTML tags can be nested.

Important: note how computers understand instructions given to them by human programmers.

HTML Attributes

< tag ATTR= ”value” > contents < /tag >

Anchors are for making links

anchor = < a >

Link example:

< a href= "www.w3c.org/html" > HTML < /a >

Images example:

src = source
alt = alternate

Text is required for broken requests. No contents to an image tag and thus no closing tag.

A void tag has no content

Whitespace: break and paragraph tags:

< br > = break (void tag)

< p > content < /p > = paragraph

< br > = inline : ends a line

Examples:

< b >
< em >
< img >
< p > = block: makes an invisible box with height and width.
